# ConvertX

Conversor simples de vídeo e áudio.

## Rodar localmente

```bash
npm install
npm start
```

Acesse:

```bash
http://localhost:3000
```

## Recursos v0.1.0

- Upload de 1 arquivo por vez
- Análise com FFprobe
- Conversão com FFmpeg
- Download do arquivo convertido
- Limpeza automática de arquivos temporários

## Formatos de saída iniciais

Vídeo: MP4, AVI, MKV, MOV, WEBM  
Áudio: MP3, AAC, WAV, OGG
